Веб-пейджер: регистрация → список людей → чат.

⚡ ВХОД В КОМАНДУ

Зачем: уникальный SS-ID (позывной). По нему сервер доставляет сообщения.

  1. Заполни поля → «Зарегистрироваться» → получишь SS-ID.
  2. Собеседник открывает ту же ссылку и регистрируется → появится в списке.
  3. Нажми на контакт → чат. Пиши и жми «Отправить».
  4. WebSocket авто-подключается; при обрыве — переподключение.
